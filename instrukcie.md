# SoulDesign.sk – Brief pre AI agenta
> Zadanie na tvorbu kompletnej webovej stránky

---

## Situácia

Si skúsený webový vývojár a dizajnér s expertízou v tvorbe moderných, responzívnych webových stránok.

Tvoríš web pre značku **SoulDesign.sk** — butikové kreatívne štúdio jednej osoby, ktoré prepája:
- moderné vibe coding weby
- grafický dizajn
- AI creative content
- reklamnú vizuálnu tvorbu
- kreatívnu online podporu značiek

Nejde o technologický startupový web.

Web musí pôsobiť:
- jemne
- elegantne
- vzdušne
- esteticky
- moderne
- žensky
- autenticky
- profesionálne bez korporátneho feelingu

**Celý web musí pôsobiť ako moderné butikové kreatívne štúdio jednej osoby.**

---

## Cieľ

Dodaj kompletnú profesionálnu mobile-first webovú stránku, ktorá bude:
- vizuálne atraktívna
- nadčasová
- UX friendly
- responzívna
- pripravená na okamžité použitie

Dôležitejšia než efekty je:
- **atmosféra**
- **typografia**
- **fotografia**
- **spacing**
- **emotional branding**
- **vizuálny pokoj**

---

## Úloha

Vytvor funkčný web, ktorý bude obsahovať:
- Štruktúrovaný komentovaný HTML5 kód s validnou sémantikou
- Responzívny dizajn (mobile-first prístup)
- CSS štýly pre prispôsobenie všetkým obrazovkám
- Moderné CSS vlastnosti (CSS variables, transitions, animations)
- CSS jednotky: texty v `rem`, nadpisy pomocou `clamp()`
- Jemný JavaScript pre interaktivitu a smooth animácie
- Ochranu formulára pomocou honeypot
- Čistý a optimalizovaný frontend kód
- SEO friendly štruktúru
- Accessibility best practices

**Používaj:**
- smooth fade animácie
- jemné hover efekty
- elegantné reveal animácie
- subtle transitions

**Nepoužívaj:**
- agresívne animácie
- prehnané efekty
- technologický SaaS vzhľad
- bootstrap feeling

## Hero sekcia

Hero sekcia je už vytvorená a nachádza sa v:
```
Web Souldesign/hero.html
```

Hero sekciu:
- **nemeň**
- **neprerábaj**
- zachovaj jej animácie
- zachovaj emotional feeling
- zachovaj cinematic reveal

**Hero sekcia je vizuálny benchmark celého webu.**

Zvyšok webu musí:
- nadväzovať jemne
- nie kontrastne
- nie templateovo

Pri tvorbe zvyšku webu zachovaj:
- rovnaký vizuálny štýl ako v hero sekcii
- rovnaký design system naprieč celým webom
- rovnaké CTA tlačidlá (štýl, gradient, border-radius, hover efekt, padding, typografia)
- rovnaké brandingové farby a gradienty
- rovnaký moderný premium feeling
- konzistentný spacing a layout systém
- rovnaký glassmorphism/overlay štýl, ak je použitý
- rovnakú typografickú hierarchiu
- rovnaký minimalistický elegantný vizuálny jazyk

**Celý web musí pôsobiť ako jeden konzistentný moderný brand systém — nie ako samostatne poskladané sekcie.**

---

## Layout

- Mobile-first layout
- Jasné oddelenie sekcií
- Asymetrické rozloženie sekcií
- Vyvážené použitie whitespace
- Intuitívna navigácia
- Sticky navigácia s jemným blur efektom
- Moderný minimalistický footer
- Smooth scrolling

**Používaj:**
- image + typography layout
- editorial blocks
- jemné layering efekty
- elegantné fotografie

**Nepoužívaj:**
- bootstrap cards everywhere
- priveľa boxov
- priveľa borderov
- priveľa ikon
- SaaS layout

---

## Vizuálna hierarchia a čitateľnosť

- Jasná typografická hierarchia
- Minimálna veľkosť textu `16px`
- Dostatočný kontrast
- `line-height: 1.5–1.8`
- Nikdy nezarovnávaj text do bloku
- Maximálna šírka textu 70% obrazovky
- Veľa whitespace, editorial spacing
- Minimalistické textové bloky

Sekcie musia dýchať, pôsobiť pokojne a mať vizuálny rytmus.

---

## Dizajn

Web musí pôsobiť: editorial, minimalisticky, jemne, nadčasovo, esteticky, moderne, vzdušne.

**Používaj:**
- asymetrické sekcie
- image/text balance
- veľké fotografie
- elegantný whitespace
- typography-first layout

**Glass efekt — iba jemne, ako detail:**
- hover overlay
- floating info box
- kontaktný formulár

**Nepoužívaj:**
- glass na všetkom
- SaaS glass cards
- heavy blur everywhere
- veľké rounded pill shapes
- príliš výrazné tiene
- tmavé heavy sekcie

**Animácie:**
- smooth fade
- subtle reveal
- jemné transitions
- elegantné opacity animácie

**Nepoužívaj:**
- agresívne hover efekty
- bouncing animácie
- technologický feeling

Jemné tiene, konzistentná ikonografia, minimalistické UI prvky, jemné radiusy `10–14px`.

## Texty

Drž sa dodaných textov doslova.

**Nemeň:**
- význam
- tone of voice
- dĺžku referencií

Môžeš upraviť iba typografiu, zalomenie, spacing, zvýraznenie.

Referencie:
- neskracuj
- zachovaj autentický text
- pri dlhej referencii použi tlačidlo „Zobraziť viac" — po kliknutí sa elegantne rozbalí celý text

Referencie sú emocionálny trust prvok — veľmi dôležitá časť webu.

## Konzistentnosť

- Jednotný štýl komponentov
- Konzistentné spacingy
- Jemné tiene
- Konzistentná ikonografia
- Minimalistické UI prvky
- Jemné radiusy `10–14px`

---

## Základné SEO

- Štrukturuj nadpisy H1–H6
- Meta title a meta description pre každú stránku
- Open Graph meta tagy
- `sitemap.xml`
- `robots.txt`
- `llms.txt`
- Alt popisky obrázkov
- Vnútorné prelinkovanie
- Structured data podľa potreby
- Všetky interné odkazy píš bez `.html`
- Do `.htaccess` pridaj clean URL pravidlá
- Zachovaj SEO hodnotu existujúcich stránok
- Zachovaj alebo priprav 301 redirecty pri zmene URL
